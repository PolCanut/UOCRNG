Help Section

Introduction

Welcome to Deathroll - RNG! This exciting 1vs1 betting game against the machine challenges your luck. Your goal is to accumulate as many coins as possible while avoiding getting the number 1.

Game Overview

You start with 50 coins. In each game, you bet 10 coins. The game is based on generating random numbers, with the range of available numbers decreasing each turn. The player who gets the number 1 loses the game.

Quick Start Guide

How to Start

  1. Download and open the game.
  2. Start a game: Press the "Play" button to begin.
  3. Initial bet: Each game has a fixed bet of 10 coins.

First Steps

Initial turn: Press the button to generate a random number between 1 and 100.

Subsequent turns: Each generated number reduces the maximum range for the next turn.

Goal: Avoid getting the number 1 and try to make the machine get it.

Game Controls

Basic Controls

Number Generation Button: Press this button to generate a random number within the available range on your turn.

Game Mechanics

Game Objectives

The goal is to accumulate as many coins as possible while avoiding losing. Turn are played alternately between you and the machine.

Rules and Regulations

  • Start of Game: The game starts with a range of numbers from 1 to 100.
  • Range Reduction: If you generate the number 50 on your turn, the next turn will have a range from 1 to 49. And so on.
  • End of Game: The game ends when one of the players gets the number 1.
  • Coins: You win 10 coins if you don't roll the number 1. You lose 10 coins if do you roll the number 1.

Game Example

First turn: You generate a 50. The range for the next turn will be 1 to 49.

Second turn (machine): The machine generates a 30. The range for the next turn will be 1 to 29.

Third turn: You generate a 15. The range for the next turn will be 1 to 14.

Fourth turn (machine): The machine generates an 8. The range for the next turn will be 1 to 7.

Continue until one of the players generates the number 1.

Frequently Asked Questions (FAQ)

Common Issues

What happens if I run out of coins? You can keep playing with a negative balance, but the challenge will be to get back to positive.

How is the random number generated? The number is generated completely randomly within the allowed range on each turn.
